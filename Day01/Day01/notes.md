# Day 01 – 2:1 Multiplexer (MUX)

- Learned how to write a **2:1 MUX** in SystemVerilog.
- Practiced using `always_comb` for combinational logic.
- used 'unique case' for condition checking 
- Inputs: `in1`, `in2`, `s`
- Output: `out`
- Behavior: 
  - If `s = 0` → `out = in1`
  - If `s = 1` → `out = in2`
